# nutrihelth — Gestão de Usuários (PHP OO + PDO)
CRUD simples com PHP orientado a objetos, PDO e MySQL.

## Instalação
1. Copie `nutrihelth` para `C:\xampp\htdocs\`
2. Inicie Apache e MySQL no XAMPP
3. No phpMyAdmin, execute `database.sql`
4. Acesse `http://localhost/nutrihelth/public`

## Observações
- A tabela utilizada é `user` (com crases no SQL), conforme solicitado.
- Campos: id, name, email, password, typeUser (A/N/U).
